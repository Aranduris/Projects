What is this database?

This database is about finding viral proteins for viruses that specifically infect certain plant species.
This information can help us understand and study the viral proteins and potentially
develop pesticides which can protect agricultural crops from viral diseases.
The plant genuses were selected using annual crop production data around the world from Food and Agricultural Organization (FAO).
Then using Plant Viruses Online website, NCBI and CDD further data was collected for this database.

Various Fuctionalities of this website:

The Browse section allows for the user to be able to fully view all the data that was gathered at once
The Search bar allows the user to be able to search for a specific plant based on scientific name of the plant by default.
To change the search parameters simply select the desired search parameter from the drop down menu.
The Common name search allows user to search for a specific plant based on common name of the plant
The Viral search allows the user to search for a specific viruse's data within the database
The motif search allows the user to search a specific morif by name to see if the motif is found withn various viruses
